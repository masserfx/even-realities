G2 AI Assistant

Real-time translation, AI chat & comic stories for Even Realities G2 smart glasses

Live Simultaneous Translation

Real-time audio translation with parallel processing pipeline Continuous microphone capture with overlapping chunk processing — while one segment is being transcribed and translated, the next is already recording. Whisper ASR + LLM translation, results display progressively on G2 HUD and are spoken via OpenAI TTS to connected earbuds. Ideal for foreign language tours, meetings, and live conversations.
Multi-language support with auto-detection Automatic source language detection via Whisper. Target languages: Czech, English, German, French, Spanish, Japanese, Chinese. Context-aware translation using sliding window of recent phrases for coherent output.
Voice Activity Detection Browser-side VAD using Web Audio API AnalyserNode. Silent chunks are skipped, saving API costs and reducing noise.

AI Chat Assistant

Streaming GPT responses on glasses HUD Voice or text input, SSE streaming responses with progressive display. Paginated text navigation via gestures. Optimized prompts for small HUD display, with automatic expansion for longer content (stories, explanations).

Comic Story Generator

AI-generated illustrated stories displayed on G2 LLM generates structured scenes with narration + image prompts. DALL-E creates black & white illustrations, processed to 2-bit indexed PNG (cnum=4) for G2's greyscale display. Synchronized TTS narration with image/text alternation.

G2 Display Integration

Optimized for 576×288px greyscale panel TextContainerProperty layout with header/content/footer. ImageContainerProperty (288×144px) with UPNG.js encoding. Paginated text (240 chars/page) with auto-scroll during streaming. Throttled updates (350ms) to prevent display flooding.

Gesture Control

Full gesture-driven interaction Click: start/stop recording, pause/resume TTS. Scroll up/down: page navigation, language switching. Double-click: mode switching (Translate / Chat / Comic). All gestures work via Even Hub SDK events.

Architecture

Mic (G2/iPhone) → 3s audio chunks → base64 ↓ POST /api/stream → Whisper ASR → GPT Translation ↓ ↓ Progressive display on G2 HUD OpenAI TTS → earbuds ↓ Multi-engine: GPT-4o-mini | GPT-4o | Ollama (offline)

Technology Stack

Even Hub SDK 0.0.9 TypeScript Hono (server) Vite (frontend) OpenAI API Whisper ASR DALL-E 2 Sharp UPNG.js Web Audio API Ollama/Gemma

Offline Capability

Local LLM fallback via Ollama Runtime engine switching between cloud (GPT-4o-mini, GPT-4o) and local (Ollama with Gemma models). Enables offline translation when internet is unavailable.
